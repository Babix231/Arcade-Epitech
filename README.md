# Arcade-Epitech
Arcade with dynamic libraries (ncurses, sfml, sdl, raylib) containing 2 games (snake, pacman).

# How to use the project
```
make all
```
> Choose the lib from the lib folder
```
./arcade ./lib/xxxx.so
```

Enjoy our Arcade !
